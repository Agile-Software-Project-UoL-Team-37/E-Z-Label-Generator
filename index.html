USER GUIDE

OVERALL

This application makes generating hundreds of unique printable labels simple, fast and efficient.
Simply enter a list of names, titles, subtexts or other information in the NAMES panel. Select properties such as colors and images to customize labels.

Then select 1 or multiple templates from the TEMPLATES panel. Modify properties such as border width, corner rounding and sizing until satisfied.

The PREVIEW page will display the generated labels. In order of templates given. On larger sessions, the user will need to manually refresh the page (red flashing button) in order to see the changes.

Note: One unique label will be generated for each name, as well as each selected templates. e.g. 10 names, and 3 templates will generate 30 labels.
Once satisfied, click on save PDF, select save as PDF and download the PDF file, ready for print.


NAMES PANEL

- This panel houses all capabilities to define unique characteristics for each label.
- On longer lists, the user will need to scroll up/down while this panel is highlighted in order to see the other names.

GLOBAL CONTROL SECTION

From left to right:

ADD ROW BUTTON: Will add a new empty row to the NAMES panel

GLOBAL ENABLED: This will toggle ALL names on and off quickly. A confirmation is required.
GLOBAL SUBTEXT: This will overwrite ALL the subtext fields for enabled rows. A confirmation is required.
GLOBAL COLOR SELECT: This will overwrite ALL color selections for enabled rows. A confirmation is required.
GLOBAL IMAGE SELECT: This will overwrite ALL image selections for enabled rows. A confirmation is required.
[Caution] GLOBAL DELETE: This will DELETE ALL enabled rows (Be careful, this step cannot be undone) A confirmation is required.

LIST SECTION

From left to right:

ID: This is the row number
Enabled: This will toggle the name on and off. Disabled names will not appear in the preview section and will not be modified by the global header controls.
Names: This is the text that will appear in the name field of the template
Subtext: This is the text that will appear in the subtext field of the template.
Color: This will open a color picker window. templates will be highlighted with this color
Image: This will open the image selection screen. Where the user can modify and select images. This image will appear in the image field of the template.
[Caution] Delete: This will delete the row.

IMAGE SCREEN

Select an image to assign to the selected row.

Upload your own image(s) by clicking on Choose files. (Square images work best)
[Caution] DELETE MODE toggle switch: Click to toggle on and off, If active, image panel will turn red.
In this mode clicking on an image will remove it from the list permanently.
NO IMAGE: Selecting this block will remove the image from the template.
RANDOM: Selecting this block will assign a random image to the selected row.


PREVIEW PANEL

- This panel merely serves as a preview for the generated templates.
- Every selected name will be generated for each selected template.
- Scroll up/down to see the different selected templates.
- The preview panel will refresh every 5 seconds.
- On larger documents, the auto-refresh will stop to prevent the slowing down of the application while many templates are being generated - the user will have to manually refresh the page.

HEADER BUTTONS

REFRESH PAGE: Clicking this will refresh the page. When the auto-refresh is disabled, the user will have to click on the flashing red button in order to refresh the page manually.
The button text will change to page up to date and turn green. This will signal the preview is up to date.

SAVE PDF: This button refreshes the page first and processes the labels ready for export.
This will open the browsers print function. Here the user can choose to print directly, but in most cases the print margins malfunction.
Instead, select the save to PDF option from the print menus dropdown menu to download the PDF. And print from the PDF directly after it is downloaded.
As this button refreshes the document first, it is not required to refresh the page before this button is pressed to update the preview.


TEMPLATES PANEL

- This panel displays and controls all the predefined templates.
- Scroll up/down with the panel selected to see more templates.
- Each template houses its own controls.

HEADER SECTION

Template enabled Checkbox: This will toggle the template on and off. If a template is turned on. Each row in the NAMES panel will be generated in the selected template.
Template Name: The template name is displayed here â€“ it will show in green if it is currently enabled. Controls will not work until template is enabled.

Controls from left to right:

Padding: This will increase or decrease the white space between the template and the cut lines. This can also be used to slightly tweak the label size.
Border: This will increase or decrease the thickness of the border around the template
Rounded: This checkbox will toggle rounded corners on and off.
Rounding: This will increase and decrease the rounding amount on the template edges. The rounded checkbox needs to be enabled for this to have an effect.
Division: This will essentially select the size of the template. The fractions are linked to the width of the page. E.g. 1 / 2 means the template will be the width of half an A4 page.


DISPLAY SECTION

This section will display a sample of the selected template. Any changes made to the control section will reflect here.
If the template is disabled, it will be dimmed to show it is not in use.
